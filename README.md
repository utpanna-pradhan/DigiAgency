# DigiAgency 🚀

A modern and responsive Digital Marketing Agency landing page built using HTML, CSS, and JavaScript.
Designed with glassmorphism UI, smooth animations, responsive layouts, and conversion-focused sections.

## 🌟 Live Preview
[click](https://utpanna-pradhan.github.io/DigiAgency/)

## ✨ Features
- Modern UI/UX Design
- Fully Responsive Layout
- Sticky Navbar
- Mobile Navigation Menu
- Hero Section with Floating Animation
- Smooth Scroll Animation
- Glassmorphism Cards
- Testimonials Section
- CTA Section
- Trust Statistics Section
- Hover Effects & Interactive UI
- Clean and Structured Code
  
## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Remix Icons
- Google Fonts (Poppins)
